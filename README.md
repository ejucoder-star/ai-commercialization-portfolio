# ai-commercialization-portfolio (Next.js)

Trilingual portfolio (ZH/EN/JA) using Next.js App Router.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Import this repo in Vercel.

## Customize
- Replace `public/resume.pdf`
- Update email/GitHub in `messages/*.json`
- Replace images under `public/images/`
